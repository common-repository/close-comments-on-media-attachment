This plugin disable/Closes comments on media attachment pages in your WordPress blog.

== Description ==

Media attachments have their own pages in your blog, and visitors can comment on it. However, most of the time these comments are unneeded and spam in most of the cases.

The plugin allows you to Disable comments on the attachment pages and get rid of spammy comments on your blog.

= Features =
* Disable/Close Comments on Media Attachment Pages

== Installation ==

1. Download and extract plugin files to wp-content/plugin directory.
2. Activate the plugin.
